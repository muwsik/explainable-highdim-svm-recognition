Модель *Base*. Генерация данных для направляющего вектора [1,0,0,...,0] происходит согласно следующей модели:
1) все объекты генерируются в гиперкубе со стороной *size*
2) первый признак объктов положительного класса имеют функцию плотности 
согласно положительной экпоненциальной функции (растущая вправо) в интервале [-*size*, +1] и 
согласно равномерному закону в интервале [+1, +*size*]
3) первый признак объктов отрицательного класса имеют функцию плотности 
согласно равномерному закону в интервале [-*size*, -1] и
согласно отрицательной экпоненциальной функции (убывающая вправо) в интервале [-1, +*size*]
4) остальные признаки распределны равномерно

Соответствие произвольному направляещему вектору осущетсявется за счет ортогонального
преобразования (поворота) модели *Base*

Положение гиперплоскости относительно начала координат задаётся параметром *offset*.
Смещение реализуется сдвигом объектов вдоль направляющего вектора.
В результатае границы гиперкуба также сдвигаются

Степень перемешенности объектов относительно гиперплоскости задаётся параметром *sigma* экспоненциальной функции

Количество объектов в каждом классе одинаковое
